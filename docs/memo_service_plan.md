# Crispy Note 서비스 기획서

## 1. 서비스 개요

### 서비스명

**Crispy Note** (크리스피 노트)

### 핵심 컨셉

"자유롭게 작성하고, 편하게 구겨서 버린다"

- 부담 없는 메모 작성: 완벽할 필요 없는 자유로운 메모장
- 가볍고 빠른 경험: 복잡한 기능 없이 본질에 집중
- 구겨서 버리기: 메모 삭제가 부담스럽지 않은 UX
- 종이 메모지의 감성을 디지털로

### 브랜드 정체성

- **컬러**: 크림/베이지 계열 (종이 느낌) + 포인트 컬러
- **톤앤매너**: 가볍고, 친근하고, 편안한
- **타겟**: 부담 없이 생각을 정리하고 싶은 모든 사람

### 핵심 가치

1. **자유로움**: 형식에 얽매이지 않는 자유로운 작성
2. **가벼움**: 복잡한 기능 없이 빠르고 간단하게
3. **편안함**: 삭제해도 괜찮은, 부담 없는 메모

## 2. 페이지 구조

### 2.1 랜딩 페이지

**히어로 섹션**

- 메인 카피: "생각을 자유롭게, 구겨서 버리듯 가볍게"
- 서브 카피: "부담 없이 끄적이고, 마음대로 버리는 당신의 디지털 메모지"
- CTA: "지금 바로 끄적이기" (회원가입 없이 바로 체험 가능)
- 배경: 살짝 구겨진 종이 텍스처 또는 메모지 애니메이션

**핵심 기능 소개**

1. **자유롭게 작성**
   - "생각나는 대로, 형식 없이 자유롭게"
   - 아이콘: 연필/펜 이미지
2. **구겨서 버리기**
   - "필요 없으면 구겨서 휴지통에"
   - 아이콘: 구겨진 종이 애니메이션
3. **어디서나 접근**
   - "PC, 모바일, 태블릿 어디서나"
   - 아이콘: 멀티 디바이스

**데모 섹션**

- 실제 메모 작성 → 구겨서 버리기 애니메이션 시연
- 인터랙티브 체험 영역

**가입 섹션**

- "당신의 메모를 저장하려면?"
- 소셜 로그인 (구글, 깃허브)
- 이메일 회원가입
- "회원가입 없이 둘러보기" 옵션 (임시 메모 체험)

---

### 2.2 로그인/회원가입

**특징**

- 간단하고 빠른 가입 절차
- 비회원도 임시로 사용 가능 (세션 기반, 브라우저 저장)

**로그인 방식**

- 소셜 로그인: 구글, 깃허브
- 이메일 + 비밀번호
- 비밀번호 찾기

**회원가입**

- 필수: 이메일, 비밀번호
- 선택: 닉네임 (기본값: 자동 생성)
- 약관 동의

### 2.3 메인 대시보드 (목록 페이지)

**레이아웃**

```
┌─────────────┬──────────────────────────────┐
│             │  헤더                        │
│             │  [검색] [새 메모+] [프로필]  │
│  사이드바   ├──────────────────────────────┤
│             │  정렬: [최신순 ▼] [카드뷰 □] │
│             ├──────────────────────────────┤
│             │  ┌─────┐ ┌─────┐ ┌─────┐    │
│             │  │메모1│ │메모2│ │메모3│    │
│             │  └─────┘ └─────┘ └─────┘    │
│             │  ┌─────┐ ┌─────┐ ┌─────┐    │
│             │  │메모4│ │메모5│ │메모6│    │
│             │  └─────┘ └─────┘ └─────┘    │
└─────────────┴─────────────────────────────┘
```

**메모 카드 (종이 메모지 스타일)**

- 살짝 구겨진 종이 느낌의 카드 디자인
- 호버 시: 살짝 들어올려지는 애니메이션
- 각 카드 구성:
  - 제목 (없으면 첫 줄 미리보기)
  - 내용 미리보기 (2-3줄)
  - 수정일 (예: "3분 전", "어제", "2024.10.07")
  - 태그 (선택)

**컨텍스트 메뉴 (우클릭 또는 길게 누르기)**

- 편집
- 복사
- 구겨서 버리기 (애니메이션과 함께 삭제)
- 태그 추가

**빈 상태**

- 일러스트: 빈 메모지
- "첫 메모를 끄적여보세요"
- "새 메모 작성" 버튼

### 2.4 사이드바

**디자인 컨셉**

- 코르크 보드 또는 책상 서랍 느낌
- 미니멀하고 직관적

**구성**

```
┌─────────────────┐
│ 📝 Crispy Note  │  ← 로고
├─────────────────┤
│ 📄 모든 메모    │
│ ⭐ 즐겨찾기     │
├─────────────────┤
│ 📁 폴더         │
│   · 아이디어    │
│   · 할 일       │
│   · 공부        │
│   [+ 새 폴더]   │
├─────────────────┤
│ 🏷️ 태그         │
│   #중요         │
│   #작업         │
│   #개인         │
├─────────────────┤
│ 🗑️ 휴지통       │
│ ⚙️ 설정         │
└─────────────────┘
```

**기능**

- 폴더: 드래그 앤 드롭으로 메모 이동
- 태그: 클릭으로 필터링
- 휴지통: 30일 후 자동 삭제
- 최소화/확장 가능

### 2.5 메모 상세/편집 페이지

**레이아웃**

```
┌─────────────────────────────────────────┐
│ [←] 제목 입력 (선택)          [자동저장] │
├─────────────────────────────────────────┤
│ 📁 폴더 선택 | 🏷️ 태그              [⭐] │
├─────────────────────────────────────────┤
│                                         │
│                                         │
│  에디터 영역                             │
│  (자유롭게 작성하세요...)               │
│                                         │
│                                         │
│                                         │
├─────────────────────────────────────────┤
│ 생성: 2024.10.07 | 수정: 방금 전        │
└─────────────────────────────────────────┘
```

**에디터 특징**

- **심플한 텍스트 에디터** (Notion 같은 복잡함 지양)
- 마크다운 경량 지원 (\*_, _, -, [ ] 정도만)
- 자동저장 (2초마다)
- 제목 생략 가능 (첫 줄이 자동으로 제목)

**간단한 포맷 툴바**

```
[B] [I] [U] [목록] [체크박스] [링크]
```

- 최소한의 포맷팅만 제공
- 필요 없으면 숨기기 가능

**빠른 액션**

- 우측 상단 메뉴:
  - ⭐ 즐겨찾기
  - 🗑️ 구겨서 버리기 (애니메이션)
  - 📤 내보내기 (TXT, MD)
  - 🔗 공유 링크 생성

**"구겨서 버리기" 애니메이션**

- 버튼 클릭 시: 메모가 구겨지는 애니메이션 (1초)
- 효과음: 종이 구기는 소리 (선택 가능)
- 휴지통으로 이동
- Toast 메시지: "메모를 구겨서 버렸어요 (되돌리기)"

## 3. 핵심 기능

### 3.1 MVP 필수 기능

**메모 관리**

- ✅ 메모 작성/수정/삭제
- ✅ 제목 자동 생성 (첫 줄)
- ✅ 자동저장
- ✅ 검색 (제목, 내용)

**조직화**

- ✅ 폴더 분류 (최대 2depth)
- ✅ 태그 시스템
- ✅ 즐겨찾기

**UX**

- ✅ "구겨서 버리기" 애니메이션
- ✅ 휴지통 (30일 보관)
- ✅ 다크모드
- ✅ 반응형 디자인

### 3.2 향후 추가 기능

**편의 기능**

- 단축키 지원 (Ctrl+N: 새 메모 등)
- 템플릿 (일기, 회의록, 할 일 등)
- 메모 색상 변경 (포스트잇처럼)
- 손글씨 입력 (태블릿)

**고급 기능**

- 메모 간 링크
- 첨부파일 (이미지만)
- 버전 히스토리
- 오프라인 모드

**소셜 기능**

- 메모 공유 (읽기 전용 링크)
- 공개 메모 갤러리 (선택적)

## 4. 사용자 시나리오

### 시나리오 1: 급하게 메모하기

1. 사이트 접속
2. "새 메모" 클릭 (또는 단축키)
3. 바로 타이핑 시작
4. 자동저장됨
5. 브라우저 탭 닫기 (저장 완료)

### 시나리오 2: 오래된 메모 정리

1. 목록 페이지에서 오래된 메모 확인
2. 메모 우클릭 → "구겨서 버리기"
3. 구겨지는 애니메이션 재생
4. 휴지통으로 이동
5. (후회되면) 휴지통에서 복구

### 시나리오 3: 메모 찾기

1. 상단 검색바에 키워드 입력
2. 실시간 검색 결과 표시
3. 태그로 필터링
4. 원하는 메모 클릭
